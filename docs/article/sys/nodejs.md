# 安装Node.js

## CentOS环境
获取资源
```bash
curl -sL https://rpm.nodesource.com/setup_8.x | bash -
```

安装
```bash
yum install -y nodejs
```

查看版本
```bash
node -v
```

安装
```bash
npm -v
```

## Window环境
window环境安装流程相对于CentOS来说比较简单，在[nodejs官网](https://nodejs.org/en/)下载最新的稳定版。运行安装包，安装过程选择默认安装即可。
